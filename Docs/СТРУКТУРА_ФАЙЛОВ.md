## 🔧 БАЗОВЫЕ ФАЙЛЫ (обязательные для работы)

### 1. settings.gs
**Назначение:** Конфигурация API ключей и URL

**Функции:**
- `OZON_API_KEY()` - API ключ Ozon
- `OZON_CLIENT_ID()` - Client ID Ozon
- `WB_API_TOKEN()` - API токен Wildberries
- `WB_API_KEY()` - API ключ Wildberries (для цен)
- `RPS()` - лимит RPS для Ozon (20)
- `WB_RPS()` - лимит RPS для WB (2)

**URL функции:**
- `ozonProductsApiURL()` - v3/product/list
- `ozonProductsInfoApiURL()` - v3/product/info/list
- `ozonAnalyticsData()` - v1/analytics/data
- `ozonPricesApiURL()` - v5/product/info/prices

**Используется всеми файлами:** ✅

---

### 2. utils.gs
**Назначение:** Вспомогательные функции

**Функции:**
- `rateLimitRPS(lastRequestTime, rps)` - контроль скорости запросов
- `retryFetch(url, options)` - HTTP запрос с повторами
- `sleep(seconds)` - пауза
- `mainSheet()` - получение листа "тест"
- `columnLetterToIndex(letter)` - буква колонки в число
- `addAndFormatRow(sheet, col, value)` - добавление строки

**Используется всеми файлами:** ✅

---

### 3. fetchapp.gs
**Назначение:** HTTP библиотека с retry логикой

**Функции:**
- `FetchApp(opts)` - основной класс для HTTP запросов
- `stopRequests()` - остановка всех запросов
- `fetch()` - выполнение запроса с повторами

**Используется через utils.gs:** ✅

## 🚀 ГЛАВНЫЕ ФАЙЛЫ (точки входа)

### 4. Главные функции.gs ⭐
**Назначение:** Главный файл с функциями для триггеров

**Основные функции:**
```javascript
// Триггерные функции (вызываются автоматически)
OzonMain()           // Ozon: товары, остатки, цены, SKU, FBS
WbMain()             // WB: остатки, заказы, цены, аналитика
OzonSKUAndAnalytic() // Ozon: аналитика (отдельно)
maintainArticleColumn() // Проверка артикулов

// Ручной запуск
all()                // Все функции подряд
testAll()            // Тест всех функций
runOzonOnly()        // Только Ozon
runWbOnly()          // Только WB
```

**Вызывает:**
- `syncOfferIdWithProductId()` - синхронизация ID
- `updateProductsV2()` - товары с брендом/моделью
- `updateStockFBO()` - остатки FBO
- `getOzonPricesOptimized()` - цены Ozon
- `updateSkuByProductId()` - SKU
- `getStocksByWarehouseFBS()` - FBS склад
- `importStocksWithImages()` - импорт WB остатков
- `updateOrdersSummaryV2()` - заказы WB
- `updatePricesAndImages()` - цены WB
- `main()` - функция WB
- `updateWBArticles()` - артикулы WB (НОВОЕ)
- `updateWBAnalytics()` - аналитика WB (НОВОЕ)

**Статус:** ✅ Обновлён с новыми функциями
