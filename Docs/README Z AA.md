# Склады Z (26) и AA (27) - Инструкция

## 📋 Что делать

### 1. Загрузить файл в Apps Script

**Файл:** `WB Склады.gs`

**Куда:** Google Sheet → Extensions → Apps Script → Создать файл "WB Склады.gs"

**Скопировать код** из файла `WB Склады.gs` в эту директорию.

---

### 2. Обновить Главные функции.gs

Уже обновлено! Функция `updateWBWarehousesByName()` добавлена в `WbMain()`.

---

### 3. Выполнить функцию

```javascript
updateWBWarehousesByName()
```

Или выполнить все WB функции:
```javascript
WbMain()
```
