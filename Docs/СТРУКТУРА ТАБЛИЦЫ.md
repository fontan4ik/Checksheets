## Примечания

### Колонки с формулами (вычисляемые)
- **W (23)**: общая сумма заказов = L (12) + N (14)

### Единицы измерения
- **Остатки**: единицы товара (шт)
- **Уход**: количество заказов (шт)
- **Цены**: рубли
- **Сумма заказов**: рубли

## API Эндпоинты

### Ozon API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `v3/product/list` | Список товаров (product_id) |
| GET | `v4/product/info/attributes` | Атрибуты (бренд, модель, SKU) |
| GET | `v4/product/info/stocks` | Остатки FBO/FBS |
| GET | `v5/product/info/prices` | Цены |
| POST | `v1/analytics/data` | Аналитика (заказы, выручка) |
| GET | `v1/product/info/stocks-by-warehouse/fbs` | FBS по складам |

### Wildberries Statistics API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/supplier/stocks` | Остатки FBO/FBW по ВСЕМ складам |
| GET | `/api/v1/supplier/orders` | Заказы за период |

**Поля ответа `/api/v1/supplier/stocks`:**
```json
{
  "warehouseId": 123456,
  "warehouseName": "Коледино (Подмосковье)",
  "supplierArticle": "22068-1",
  "nmId": 216675685,
  "barcode": "...",
  "quantity": 10
}
```

### Wildberries Suppliers API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v3/warehouses` | Список складов продавца (FBS/DBS) |
| GET | `/api/v3/stocks/{warehouseId}` | Остатки на складе продавца |

**Поля ответа `/api/v3/warehouses`:**
```json
{
  "id": 123456,
  "name": "Коледино",
  "address": "Московская область,...",
  "type": "FBS"
}
```

## Функции для выполнения

### Основные функции
```javascript
all()              // Все функции Ozon + WB
OzonMain()         // Только Ozon
WbMain()           // Только WB
```

### Ozon функции
```javascript
syncOfferIdWithProductId()     // U (21): Product_id
updateProductsV2()             // C, D, E, V, X, Y: Товары
updateStockFBO()               // F (6): Остаток ФБО
updateAllFBSStocks()           // G (7): Все FBS
getStocksByWarehouseFBS()      // H (8): FBS Москва
getOzonPricesOptimized()       // K (11): Цены
fetchAndWriteAnalytics()       // I, J, L: Аналитика
```

### WB функции
```javascript
updateWBStocksFromStatisticsAPI()  // O (15): FBO WB
importStocksWithImages()           // Импорт с картинками
updateOrdersSummaryV2()            // N (14): Заказы WB
updatePricesAndImages()            // M (13): Цены WB
main()                             // O, P, FBO/FBS
updateWBArticles()                  // T (20): Артикул WB
updateWBAnalytics()                // R, S: Аналитика WB
updateWBFBSMoscow()                 // Q (17): FBS Москва
```

### Новые функции (для складов)
```javascript
updateWBWarehouses()    // Z, AA: Фильтр по warehouseName
updateWBSupplierWarehouses()  // AB, AC: FBS склады
updateFeronStocks()     // AI, AJ, AK: Feron API склады
updateETMStocks()       // AL: ETM Самара
updateExternalAPIStocks()  // Все внешние API (Feron + ETM)
```
