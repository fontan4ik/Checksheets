## 🔍 Обнаруженные проблемы

### Проблема #1: Суффикс `-1` в артикулах

**Найдено:** ETM API возвращает 404 для артикулов с суффиксом `-1`

| Артикул | Результат |
|---------|-----------|
| 41580-1 | ❌ 404 Товар не найден |
| 41580 | ✅ 200 OK |

**Решение:** Убрать суффикс `-1` перед запросом к API

---

### Проблема #2: Дублирование остатков в ответе API

**Найдено:** ETM API возвращает ДВЕ записи для склада Стройкерамика с **одинаковым остатком**:

#### Пример: MKP42-N-02-30-20

| StoreCode | StoreType | StoreName | StoreQuantRem |
|-----------|-----------|-----------|---------------|
| 13 | **rc** | Самарская область... Стройкерамика... | 1349 |
| 24121 | **op** | Самарcкая обл., Стройкерамика пгт.... | 1349 |
| - | **Сумма** | - | **2698** (2x от ожидаемого!) |

#### Сравнение всех тестовых артикулов:

| Артикул | StoreCode 13 (rc) | StoreCode 24121 (op) | Сумма | Ожидается | Отношение |
|---------|-------------------|----------------------|-------|-----------|-----------|
| MKP42-N-02-30-20 | 1349 | 1349 | 2698 | 1362 | 1.98x |
| MAD22-5-016-C-30 | 5904 | 5904 | 11808 | 6027 | 1.96x |
| UZA-11-D01-D10 | 8793 | 8793 | 17586 | 8980 | 1.96x |
| RBD-80 | 662 | 662 | 1324 | 703 | 1.88x |

**Вывод:** Склады `rc` и `op` дублируют один и тот же остаток. Ожидаемое значение ≈ одной из записей.

**Решение:** Учитывать только **StoreType="rc"** (распределительный центр)

## ✅ Внесённые исправления

### Файл: ETM API.gs

#### Изменение 1: Удаление суффикса `-1` (строка 63-65)

```javascript
// Было:
function fetchETMStockForArticle(article, sessionId) {
  const url = `https://ipro.etm.ru/api/v1/goods/${encodeURIComponent(article)}/remains?type=mnf&session-id=${sessionId}`;

// Стало:
function fetchETMStockForArticle(article, sessionId) {
  // Убираем суффикс -1 если он есть (ETM использует артикулы без -1)
  const etmArticle = article.endsWith('-1') ? article.slice(0, -2) : article;
  const url = `https://ipro.etm.ru/api/v1/goods/${encodeURIComponent(etmArticle)}/remains?type=mnf&session-id=${sessionId}`;
```

#### Изменение 2: Фильтр только StoreType="rc" (строки 91-101)

```javascript
// Было:
data.data.InfoStores.forEach(store => {
  const storeName = store.StoreName || "";
  if (storeName.toLowerCase().includes('стройкерамика')) {
    stock += (store.StoreQuantRem || store.StockRem || 0);
  }
});

// Стало:
data.data.InfoStores.forEach(store => {
  const storeName = store.StoreName || "";
  const storeType = store.StoreType || "";
  // Проверяем, что это Стройкерамика И тип "rc" (распределительный центр)
  if (storeName.toLowerCase().includes('стройкерамика') && storeType === 'rc') {
    stock += (store.StoreQuantRem || store.StockRem || 0);
  }
});
```

## 📝 Дополнительная информация

### StoreType значения:
- **rc** = Распределительный Центр (Distribution Center) - основной склад
- **op** = Операционный склад (Operational store) - может дублировать данные
