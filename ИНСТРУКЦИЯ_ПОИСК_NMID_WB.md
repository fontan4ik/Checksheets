# ИНСТРУКЦИЯ: Как найти реальный nmId в личном кабинете Wildberries

## Проблема

В таблице указан nmId **465019000** для артикула **48724-1**, но API не может найти этот товар. Это означает одно из:

1. ❌ nmId в таблице устарел/неверен
2. ❌ API токен от другого аккаунта, где нет этой карточки
3. ❌ Карточка архивирована/удалена
4. ❌ Артикул "48724-1" не соответствует этому nmId

---

## Способ 1: Через страницу карточки товара (НАДЕЖНЫЙ)

### Шаг 1: Откройте карточку товара

1. Зайдите в личный кабинет WB: https://seller.wildberries.ru/
2. В левом меню выберите **"Каталог"** → **"Все карточки"**
3. В списке найдите товар с артикулом **"48724-1"**

### Шаг 2: Нажмите на название товара

- Кликните на название карточки (не на чекбокс!)
- Откроется страница редактирования карточки

### Шаг 3: Найдите nmId в адресной строке

В адресной строке браузера вы увидите URL вида:
```
https://seller.wildberries.ru/products/edit/card/#/nmid/12345678/sku/98765432
```

**nmId** - это первое число в URL после `/nmid/`

> **Пример:** Если URL `.../nmid/98765432/...`, то nmId = **98765432**

---

## Способ 2: Через столбец "nmId" в списке карточек

### Шаг 1: Откройте список всех карточек

1. **"Каталог"** → **"Все карточки"**
2. Убедитесь что вы находитесь на вкладке **"Активные"**

### Шаг 2: Добавьте столбец "Артикул" если не видите

1. Нажмите на иконку **"Настройки таблицы"** (справа от поиска, выглядит как шестеренка или список)
2. Включите галочку **"Артикул"** (или "vendor_code")
3. Включите галочку **"nmId"** (или "WB ID")
4. Нажмите **"Применить"**

### Шаг 3: Найдите товар по артикулу

1. Используйте поиск сверху - введите **"48724-1"**
2. В столбце **"nmId"** или **"WB ID"** будет указан реальный nmId
3. Он будет выглядеть как 7-8 значное число, например: **12345678**

---

## Способ 3: Через выгрузку Excel (БЫСТРЫЙ)

### Шаг 1: Выгрузите список карточек

1. **"Каталог"** → **"Все карточки"**
2. Нажмите кнопку **"Выгрузить"** (обычно右上角)
3. Выберите **"Excel"** или **"CSV"**

### Шаг 2: Откройте файл

В выгрузке будут столбцы:
- **Артикул** (или `vendor_code`) - ваш "48724-1"
- **nmId** (или `wb_id`) - реальный ID карточки на WB

### Шаг 3: Найдите нужную строку

1. Откройте файл в Excel
2. Найдите строку с артикулом **"48724-1"**
3. В столбце **nmId** будет реальный ID

## Как проверить что API токен от правильного аккаунта

### Проверка 1: Сравните количество карточек

1. Посмотрите сколько карточек в личном кабинете:
   - Откройте **"Каталог"** → **"Все карточки"**
   - Сверху будет показано "Всего: XXX карточек"

2. Проверьте через API:
```javascript
// В Apps Script
function testWBCardCount() {
  const response = fetchApp.fetch(
    'https://content-api.wildberries.ru/content/v2/get/cards/list',
    {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + Settings.WB_TOKEN(),
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        limit: 1
      })
    }
  );

  const data = JSON.parse(response.getContentText());
  console.log('Всего карточек:', data.data?.cards?.total || 'Ошибка');
}
```

**Если цифры совпадают** → токен от правильного аккаунта ✅

### Проверка 2: Найдите любой другой товар

1. В личном кабинете найдите любой активный товар
2. Запомните его nmId (из столбца или URL)
3. Проверьте через API:

```javascript
function testWBCheckNmId(nmId) {
  const response = fetchApp.fetch(
    'https://content-api.wildberries.ru/content/v2/get/cards/list',
    {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + Settings.WB_TOKEN(),
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        settings: {
          cursor: {
            limit: 1000
          },
          filter: {
            withPhoto: false
          }
        }
      })
    }
  );

  const data = JSON.parse(response.getContentText());
  const cards = data.data?.cards || [];

  const found = cards.find(c => c.nmID == nmId);
  console.log('Карточка найдена:', !!found);
}
```

**Если найдена** → токен работает ✅

## Что делать если карточка не находится

### Ситуация 1: Карточка в архиве

1. В личном кабинете: **"Каталог"** → **"Все карточки"**
2. Переключитесь на вкладку **"Архивные"**
3. Найдите артикул "48724-1"
4. Проверьте nmId

### Ситуация 2: Карточка в корзине

1. В личном кабинете: **"Каталог"** → **"Все карточки"**
2. Переключитесь на вкладку **"Корзина"**
3. Проверьте nmId

### Ситуация 3: Карточка удалена

Если карточки нет нигде (активные/архивные/корзина):
- **Вариант 1:** Карточка была удалена окончательно
- **Вариант 2:** Карточка в другом аккаунте WB (если у вас несколько магазинов)
- **Вариант 3:** Артикул "48724-1" написан с ошибкой

## Важные замечания

### nmId vs Артикул продавца

- **Артикул продавца** (например, "48724-1") - это ваш внутренний код
- **nmId** - это ID карточки в системе Wildberries (автоматически присваивается WB)
- **Одному артикулу всегда соответствует один nmId**
- **Но nmId может измениться если карточка удалена и создана заново**

### Почему в таблице может быть неверный nmId

1. Карточка была удалена и создана заново → новый nmId
2. Ручная ошибка при копировании nmId
3. nmId скопировали из другой карточки
4. Карточка переехала в другой аккаунт

## После нахождения реального nmId

### Вариант 1: Обновить вручную

1. В Google Sheet найдите строку с артикулом "48724-1" (столбец A)
2. В столбце **T (Артикул ВБ)** замените **465019000** на реальный nmId
3. Запустите функции обновления данных

### Вариант 2: Пересчитать через API

Создайте функцию для поиска nmId по артикулу:

```javascript
function findNmIdByVendorCode(vendorCode) {
  const response = fetchApp.fetch(
    'https://content-api.wildberries.ru/content/v2/get/cards/list',
    {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + Settings.WB_TOKEN(),
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        settings: {
          cursor: {
            limit: 1000
          },
          filter: {
            withPhoto: false
          }
        }
      })
    }
  );

  const data = JSON.parse(response.getContentText());
  const cards = data.data?.cards || [];

  const found = cards.find(c => c.vendorCode === vendorCode);
  return found ? found.nmID : null;
}

// Тест
function testFindNmId() {
  const nmId = findNmIdByVendorCode('48724-1');
  console.log('Реальный nmId:', nmId);
}
```
